# vue2 手机端多页面项目

---

### 介绍

vue2-mulit-mobile 是一个手机端 vue2 多页面框架，更适用于 h5 活动页

## 安装及使用

* 安装
```bash
# 通过 pnpm 安装
pnpm i
```
* 使用
  `entry` 目录下创建入口，在 `package.json` 创建打包命令

## 使用到的官方生态

| 项目 | 描述 |
| --- | --- |
| [vue2](https://v2.cn.vuejs.org/) | vue2 官方文档 |
